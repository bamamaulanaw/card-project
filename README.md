# Card Project

A Flutter project that demonstrates various card-based UI components using Material Design.

## Description

This project focuses on layout composition and visual presentation without backend integration.

## Technologies

- Flutter
- Dart
- Material Design

## Features

- Menu cards with icons
- Food product cards
- Article style cards
- Weather information cards
- List tile menu cards
- Local asset usage

## Installation

1. Clone this repository:

```bash
git clone https://github.com/bamamaulanaw/card-project.git
```

2. Run:

```bash
flutter pub get
```

3. Run the project using:

```bash
flutter run
```
